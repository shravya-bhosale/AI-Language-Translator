# AI Language Translator

## Description
AI Language Translator is a Python and Streamlit-based web application that translates text into multiple languages. It provides a simple and user-friendly interface for quick and accurate translations.

## Features
- Translate text into multiple languages
- Simple and interactive user interface
- Fast and accurate translation
- Easy to use

## Technologies Used
- Python
- Streamlit
- Deep Translator

## How to Run
1. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   streamlit run translator.py
   ```
